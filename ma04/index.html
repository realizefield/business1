MA04　分数をたす・ひく（2）分母が異なる → 通分

【例題】次の計算をしましょう。

\(\dfrac{1}{3} + \dfrac{1}{2}\)

※分母をそろえることを「通分」といいます。

解）分母を 6 にします。 → 3 と 2 の最小公倍数

\(\dfrac{1}{3} + \dfrac{1}{2} = \dfrac{1\times2}{3\times2} + \dfrac{1\times3}{2\times3} = \dfrac{2}{6} + \dfrac{3}{6} = \dfrac{5}{6}\)

おたがいの分母をかける。

【類題】以下の計算をしなさい。

(1)\ \(\dfrac{1}{2} + \dfrac{1}{5}\)\　 (2)\ \(\dfrac{1}{3} + \dfrac{1}{5}\)\　 (3)\ \(\dfrac{2}{3} + \dfrac{2}{5}\)
(4)\ \(\dfrac{2}{5} + \dfrac{1}{7}\)\　 (5)\ \(\dfrac{1}{4} + \dfrac{2}{5}\)\　 (6)\ \(\dfrac{2}{7} + \dfrac{3}{8}\)

【類題の答え】

(1) 7/10, (2) 8/15, (3) 16/15,
(4) 19/35, (5) 13/20, (6) 37/56

【確認問題】

(1)\ \(\dfrac{3}{4} + \dfrac{1}{5}\)\　 (2)\ \(\dfrac{4}{7} + \dfrac{1}{3}\)

【確認問題の答え】

(1) 19/20, (2) 19/21
