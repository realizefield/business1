MA06　整数と分数の計算（たす・ひく）

【例題】(1)\ 2 + \(\dfrac{5}{6}\)\　(2)\ 3 − \(\dfrac{3}{7}\)\ を計算しなさい。

解）(1)\ \(2 + \dfrac{5}{6} = \dfrac{2}{1} + \dfrac{5}{6} = \dfrac{12}{6} + \dfrac{5}{6} = \dfrac{17}{6}\)

(2)\ \(3 - \dfrac{3}{7} = \dfrac{3}{1} - \dfrac{3}{7} = \dfrac{21}{7} - \dfrac{3}{7} = \dfrac{18}{7}\)

通分＝分母をそろえます。

【類題】以下の計算をしなさい。

(1)\ \(4 - \dfrac{3}{8}\)\　 (2)\ \(5 - \dfrac{1}{6}\)\　 (3)\ \(2 - \dfrac{2}{3}\)\　 (4)\ \(3 + \dfrac{3}{8}\)\　 (5)\ \(5 + \dfrac{1}{6}\)\　 (6)\ \(2 - \dfrac{2}{3}\)

【類題の答え】

(1) 29/8, (2) 29/6, (3) 4/3, (4) 27/8, (5) 31/6, (6) 4/3

【確認問題】

(1)\ \(6 - \dfrac{5}{8}\)\　 (2)\ \(4 + \dfrac{7}{9}\)

【確認問題の答え】

(1) 43/8, (2) 43/9
