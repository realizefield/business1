MA08　仮分数を帯分数にする

【例題】\(\dfrac{13}{5}\) を帯分数にしなさい。

\(13 = 5 + 5 + 3\) なので \(\dfrac{13}{5} = \dfrac{5 + 5 + 3}{5} = \dfrac{5}{5} + \dfrac{5}{5} + \dfrac{3}{5} = 1 + 1 + \dfrac{3}{5} = 2 + \dfrac{3}{5} = 2\dfrac{3}{5}\)

→ このように計算するのは面倒です。つぎのようにします。

\[ \begin{array}{c} 2 \\ \hline 5 ) 13 \\ 10 \\ \hline 3 \end{array} \ \ \rightarrow\ 2\dfrac{3}{5} \]

【類題】次の仮分数を帯分数にしなさい。

(1)\ \(\dfrac{21}{5} = 4\dfrac{1}{5}\)\　 (2)\ \(\dfrac{37}{7} = 5\dfrac{2}{7}\)\　 (3)\ \(\dfrac{58}{11} = 5\dfrac{3}{11}\)
(4)\ \(\dfrac{29}{8} = 3\dfrac{5}{8}\)\　 (5)\ \(\dfrac{49}{6} = 8\dfrac{1}{6}\)\　 (6)\ \(\dfrac{83}{10} = 8\dfrac{3}{10}\)

【確認問題】

(1)\ \(\dfrac{46}{7} = 6\dfrac{4}{7}\)\　 (2)\ \(\dfrac{35}{6} = 5\dfrac{5}{6}\)
